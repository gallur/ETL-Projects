ETL and Analytics Dashboard for Self Created Android Application

ETL Pipeline for Student Data Using Serverless Architecture and DynamoDB

Tech Stack: AWS Lambda, DynamoDB, S3, Python, Boto, CloudWatch, JSON, NoSQL.

Description: Built a data pipeline that starts with AWS S3 that triggers a lambda function which in turn parses data from the Event object and drops it into a corresponding DynamoDB table for later use.
